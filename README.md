R&D Engineer at **Expleo France**. Previously postdoctoral fellow at **INRIA Bordeaux**
(2022–2024); PhD, Université du Littoral Côte d'Opale (2021).

I work across machine learning, deep learning, and signal processing — methods and systems
that have to hold up on real, messy data. Recent focus: **transformers, generative models,
and large language models**, and the question of making them work efficiently with less data
and less compute.

### Areas

- Deep learning · CNNs · transformers · graph neural networks
- Large language models · generative models · representation learning
- Randomized numerical linear algebra · random projections
- Matrix and tensor factorization · source separation
- Inverse problems · image and signal reconstruction

### Selected publications

- **F. Yahaya** *et al.*, "A framework for compressed weighted nonnegative matrix
  factorization," *IEEE Transactions on Signal Processing*, 2024.
- **F. Yahaya** *et al.*, "Random projection streams for (weighted) nonnegative matrix
  factorization," *ICASSP*, 2021.
- **F. Yahaya** *et al.*, "How to apply random projections to nonnegative matrix
  factorization with missing entries?," *EUSIPCO*, 2019.
